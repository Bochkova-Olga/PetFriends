Посмотрим, как будут меняться состояния изменяемых файлов. Откройте один из добавленных в репозиторий файлов и измените его. Например, добавьте в него ещё одну строку. Выполните команду git status.

На экране появится информация о том, что:

Файл модифицирован.
Изменения не подготовлены для коммита.
